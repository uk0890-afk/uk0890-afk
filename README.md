# uk0890-afk
This project is a Unity prototype inspired by common mobile game mechanics.
 Key Features

    Character Selection: Choose your playable character from a roster.

    Status & Attributes: Comprehensive system for character parameters and information.

    Stages: Foundation for stage-based gameplay levels.

    Gacha System: Item randomized summoning system.

        Debug Feature: Includes a function to reset gacha currency for testing purposes.

 Installation Guide

Note: Due to external dependencies, please follow these steps exactly to avoid import errors.

    Download the Package

        Visit the repository: uk0890/practice

        Navigate to My project.unitypackage and download it.

    Prerequisites (Crucial)
    You must install the following dependencies in your Unity project before importing the package. Otherwise, the import will fail.

        UniVRM 0.66.0: Download here

        Shapes2D: Add this asset from the Unity Asset Store.

            Shapes2D - Procedural Sprites and UI

    Importing

        Once the dependencies above are installed, drag and drop My project.unitypackage into your Unity project to complete the setup.




このプロジェクトは、一般的なスマートフォン向けゲームの基本システムを参考に制作したUnityプロジェクトです。
 主な機能

    キャラクター選択: 複数のキャラクターから操作キャラを選択可能。

    ステータス管理: キャラクターごとの詳細なパラメータと情報を保持。

    ステージ構成: ゲームプレイの土台となるステージシステム。

    ガチャ機能: ゲーム内アイテムの抽選システム。

        デバッグ機能: テスト用にガチャ石の数をリセットできる機能を搭載しています。

 インストール方法

※重要：依存関係があるため、以下の手順を必ず守ってください。

    パッケージのダウンロード

        uk0890/practice へアクセス。

        My project.unitypackage を選択し、ダウンロードしてください。

    前提条件（重要）
    パッケージをインポートする前に、必ず以下の2つをプロジェクトに導入してください。これらがないとインポート時にエラーが発生します。

        UniVRM 0.66.0: ダウンロードはこちら

        Shapes2D: Unity Asset Storeから追加してください。

            Shapes2D - Procedural Sprites and UI

    インポート

        上記の依存関係を導入した空のUnityプロジェクトに、ダウンロードした My project.unitypackage をドラッグ＆ドロップしてインポートしてください。
